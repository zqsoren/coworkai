# Zzzap - 产品项目报告

## 1. 产品简介 (Product Introduction)
**Zzzap** 是一款专为**建筑设计与室内设计领域**打造的智能化 SaaS 平台。它结合了最新的**生成式 AI (AIGC)** 与 **计算机视觉技术**，旨在帮助设计师快速生成概念图、智能识别图纸信息，并提供专业的在线编辑工具，从而大幅提升设计效率，缩短从灵感到成图的周期。

该平台采用**Freemium (免费增值)** 商业模式，通过积分系统与会员订阅实现商业化，既服务于个人设计师，也具备扩展为团队协作工具的潜力。

## 2. 核心功能介绍 (Core Features)

### 2.1 🎨 AI 灵感生成器 (AI Generator)
这是产品的核心创作引擎，支持多模态输入与多样化风格输出。
*   **多级风格库**: 内置三级分类体系（分类 -> 子类 -> 风格），用户可快速选择如"现代极简"、"新中式"等特定建筑风格。
*   **高级参数配置**: 支持自定义提示词 (Prompt)、画面比例、参考图 (Reference Image) 及提示图 (Hint Image) 上传，精准控制生成结果。
*   **三大生图模式 (Generation Modes)**:
    1.  **模式 1 (Optimization): 优化+生图模式 (默认)**。最智能的模式。系统会将用户的各项参数组装，先通过文本大模型（如 GPT）进行智能润色与 JSON 重构，生成完美的指令后再调用生图 API，包容度最高。
    2.  **模式 2 (Direct): 直接生图模式**。高阶用户模式。完全跳过 AI 提示词润色，将用户输入作为变量严格按照后台设定的模板执行替换，并强制使用官方后台配置的参考图进行生图，追求绝对的稳定性与可控性。
    3.  **模式 3 (Mode C): 直出+文件名注入模式**。进阶模式。在模式 2 的基础上，自动提取用户上传的所有图片的文件名，并注入到提示词模板中。让大模型在生图时，能直接将特定图像与特定长文件名建立语义联系，适合批量化或特定物料对应的生图任务。

### 2.2 👁️ 智能图纸识别与编辑 (Smart Editor & Recognition)
基于 Fabric.js 开发的专业级在线图像编辑器，集成前沿 AI 视觉能力。
*   **AI 智能识别 (Smart Recognition)**: 集成 GPT-4o 视觉模型，能自动识别建筑图纸中的**功能分区标签**（如"主卧"、"客厅"）及布局结构，并将其转换为可编辑的组件。
*   **OCR 文字提取**: 内置 Tesseract.js 引擎，支持本地化的高精度文字识别。
*   **专业编辑工具**: 提供图层管理、画笔、通过 Canvas 进行的矢量编辑（移动、缩放、旋转），支持对生成图片进行二次精修。

### 2.3 💰 商业化闭环系统 (Commercial System)
完善的支付与会员体系，确保平台的商业可持续性。
*   **积分体系 (Credits)**: 用户通过充值或邀请好友获取积分，每次生成消耗特定积分（普通/高级/4K）。
*   **会员等级**: 自动化的会员成长体系（Free -> Pro -> Ultra），根据充值金额自动晋升，解锁更多权益。
*   **支付集成**: 支持多种支付方式记录，后台配备人工审核机制，保障资金安全。

### 2.4 👥 用户与邀请裂变 (User Growth)
*   **邀请机制**: 每位用户拥有独立邀请码，邀请双方均可获得积分奖励，促进用户自发裂变增长。
*   **数据看板**: 用户个人中心提供详细的消耗统计与充值记录，透明化管理资产。

## 3. 网站页面架构 (Site Architecture)

平台采用清晰的**左右分栏布局**或**顶部导航结构**，主要包含以下核心模块：

### 3.1 用户端 (User Facing)
| 页面模块 | 功能描述 | 对应组件/路由 |
| :--- | :--- | :--- |
| **工作台 (Dashboard)** | 用户的核心概览页，展示常用工具入口、最新动态及个人数据摘要。 | `Dashboard.tsx` |
| **灵感生成 (Generator)** | 核心功能区。左侧为参数配置面板（风格、提示词、参考图），右侧为生成结果预览与历史记录。 | `Generator.tsx` |
| **项目管理 (Projects)** | 用户的数字资产库。管理已生成的图片、收藏的项目及上传的素材，支持创建文件夹归档。 | `Projects.tsx` / `ProjectDetail.tsx` |
| **智能编辑 (Editor)** | 全屏编辑器。提供画布交互、AI 识别按钮、图层控制面板及工具栏。 | `Editor.tsx` / `ImageFocusEditorModal.tsx` |
| **个人中心 (Profile)** | 用户信息管理。包含头像修改、密码重置、邀请码查看及会员状态展示。 | `Profile.tsx` |
| **充值中心 (Recharge)** | 购买积分与会员服务的收银台页面，展示充值套餐与支付方式。 | `Recharge.tsx` |

### 3.2 管理端 (Admin Panel)
*仅管理员可见，用于平台运营与监控*
| 页面模块 | 功能描述 |
| :--- | :--- |
| **运营仪表盘** | 全局数据监控（总用户数、日活、总生成量、营收统计）。 |
| **用户管理** | 用户列表查询、封禁/解封账号、手动调整用户积分。 |
| **订单审核** | 审核用户的充值请求，处理通过/驳回逻辑。 |
| **风格管理** | 配置生成器的风格库（上传预览图、调整提示词模板、设置精选）。 |
| **系统设置** | 动态调整全局参数（如单次生成消耗积分数、邀请奖励额度）。 |

## 4. 技术亮点 (Technical Highlights)
*   **高性能前端**: 基于 React 19 + Vite 构建，秒级响应，操作流畅。
*   **无缝 AI 集成**: 前端直接集成 OpenAI 能力，实现"所见即所得"的智能识别体验。
*   **稳健后端**: Node.js + SQLite 轻量级架构，配合自动备份与迁移脚本，部署维护成本低且数据安全。
*   **自动化部署**: 配备 PowerShell 自动化部署脚本，支持一键更新服务器，降低运维门槛。
